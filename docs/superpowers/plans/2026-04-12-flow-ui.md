# Flow UI Implementation Plan

> **For agentic workers:** Use superpowers:subagent-driven-development to execute.

**Goal:** Ship a React+Vite+TS UI (Ant Design primary, minimal Tailwind) for editing and running flows, embedded into the Go binary and packaged as a single Docker image.

**Architecture:** SPA under `ui/`, built into `ui/dist`, copied into `internal/web/dist/` at Docker-build time and embedded via `//go:embed`. Go serves `/api/*` as before; all other paths fall through to the SPA with `index.html` fallback.

**Tech Stack:** React 18, Vite, TypeScript, React Flow v12, Ant Design v5, Tailwind (utilities only), React Router v6.

Spec: `docs/superpowers/specs/2026-04-12-flow-ui-design.md`.

## Task 10: Go embed + SPA handler

**Files:** `internal/web/web.go`, `internal/web/web_test.go`, `internal/web/dist/.gitkeep`, `internal/web/dist/index.html` (tiny placeholder so build works without UI).

- [ ] `web.go`:
```go
package web

import (
    "embed"
    "io/fs"
    "net/http"
    "strings"
)

//go:embed all:dist
var distFS embed.FS

func Handler() http.Handler {
    sub, _ := fs.Sub(distFS, "dist")
    fileServer := http.FileServer(http.FS(sub))
    return http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
        if strings.HasPrefix(r.URL.Path, "/assets/") {
            w.Header().Set("Cache-Control", "public, max-age=31536000, immutable")
            fileServer.ServeHTTP(w, r)
            return
        }
        f, err := sub.Open(strings.TrimPrefix(r.URL.Path, "/"))
        if err == nil {
            f.Close()
            fileServer.ServeHTTP(w, r)
            return
        }
        // SPA fallback
        index, err := sub.Open("index.html")
        if err != nil {
            http.NotFound(w, r)
            return
        }
        defer index.Close()
        w.Header().Set("Content-Type", "text/html; charset=utf-8")
        io.Copy(w, index) // add import
    })
}
```

  - falls back to index.html for unknown paths (`/flows/abc`).
  - returns 404 for unknown `/assets/*`.
- [ ] Wire into `internal/api/router.go`: `r.Handle("/*", web.Handler())` at the end.

## Task 11: Dockerfile + compose

**Files:** `Dockerfile`, `docker-compose.yml` (modify).

- [ ] Multi-stage `Dockerfile`:
```dockerfile
# UI build
FROM node:20-alpine AS ui
WORKDIR /ui
COPY ui/package*.json ./
RUN npm ci
COPY ui/ ./
RUN npm run build

# Go build
FROM golang:1.25-alpine AS api
WORKDIR /src
RUN apk add --no-cache git
COPY go.* ./
RUN go mod download
COPY . .
COPY --from=ui /ui/dist ./internal/web/dist
RUN CGO_ENABLED=0 GOOS=linux go build -trimpath -ldflags="-s -w" -o /observer ./cmd/observer

# Runtime
FROM gcr.io/distroless/static:nonroot
COPY --from=api /observer /observer
COPY migrations /migrations
EXPOSE 8080
ENV MIGRATIONS_DIR=/migrations
USER nonroot:nonroot
ENTRYPOINT ["/observer"]
```

- [ ] Update `docker-compose.yml` to add an `app` service:
